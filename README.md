# Netflix Content Strategy Analysis

An exploratory analysis of Netflix's global content catalog using Python, pandas, and Streamlit.

## Key Findings
- **US and India dominate** Netflix's catalog by volume, dwarfing all other countries combined
- **US content skews heavily adult** — TV-MA and R ratings account for nearly 50% of US titles, reflecting Netflix's strategy to compete with premium cable
- **TV-MA is the most common rating globally**, suggesting Netflix targets adult audiences across all major markets
- **Netflix's genre taxonomy is inconsistent** — "International Movies" is an origin-based tag while "Dramas" is content-based, causing overlap and making pure genre analysis unreliable
- **Content addition peaked in 2019**, with Movies consistently outnumbering TV Shows. The post-2020 decline aligns with COVID-19 production shutdowns

## Tools Used
- Python, pandas, matplotlib, seaborn, Streamlit
